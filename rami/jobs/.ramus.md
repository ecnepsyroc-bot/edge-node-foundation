# Ramus: Jobs

## Purpose
Job listing management and validation.

## Responsibilities
- Maintain list of 34 predefined jobs
- Validate job names
- Validate subcategory (Factory Order) names
- Validate individual chat names
- Check job existence
- Provide default subcategory name

## Public Interface

### Constructor
```javascript
new JobsManager()
```

### Methods
- `getAllJobs()` - Get array of all job names
- `addJob(jobName)` - Add new job (validates via SAP)
- `jobExists(jobName)` - Check if job exists
- `validateSubcategoryName(name)` - Validate Factory Order name
- `validateChatName(name)` - Validate individual chat name

## Dependencies
- `water/schemas.js` (predefined jobs list, defaults)
- `sap/validators.js` (validation functions)

## Does NOT Handle
- Message storage
- User management
- HTTP/WebSocket communication
- Subcategory creation (delegated to ChatManager via graft)

## Integration Points
- Connected to ChatManager via `chat-jobs` graft
- Uses WATER schemas for job definitions
- Uses SAP validators for name validation

## Predefined Jobs (34 total)
Includes: General, 3410 Marpole Avenue, Archetype, Bellamy, Blueberry, Cambie Club Station, Casino Club Harvest, etc.
